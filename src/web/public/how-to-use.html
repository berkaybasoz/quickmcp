üöÄ Quick Setup for Claude Desktop

1

Make sure QuickMCP is running

Keep this application running with npm run dev

Checking server status...
2

Add to Claude Desktop Config

Copy this configuration to your Claude Desktop config file:

{
  "mcpServers": {
    "quickmcp-integrated": {
      "command": "curl",
      "args": ["-X", "POST", "http://localhost:3001/sse/message"],
      "env": {
        "MCP_TRANSPORT": "sse"
      }
    }
  }
}
3

Restart Claude Desktop

Close and reopen Claude Desktop for the changes to take effect.

üöÄ Export Process for Claude Desktop

1

Go to Manage Servers

Navigate to the Manage Servers page.

2

Export Your Server

Click the "Export" button for the server you want to use.

3

Extract and Install

Extract the downloaded package and run npm install in the extracted folder.

4

Configure Claude Desktop

Add the server to your Claude Desktop config:

{
  "mcpServers": {
    "your-server-name": {
      "command": "node",
      "args": ["path/to/your/server/index.js"]
    }
  }
}
