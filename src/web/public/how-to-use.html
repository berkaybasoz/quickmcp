🛠️ Using Your Tools

In Claude Desktop, your tools will be available with this naming pattern:

serverName__toolName

Example: If you have a server called "sales-data" with a tool called "search_customers", it will appear as sales-data__search_customers

Individual Export Method

Export and configure each server separately

✅ Advantages

  • Independent server management
  • Clean tool naming without prefixes
  • Better performance for individual servers
  • Portable server packages

🚀 Export Process for Claude Desktop

1
Go to Manage Servers

Navigate to the Manage Servers page.

2
Export Your Server

Click the "Export" button for the server you want to use.

3
Extract and Install

Extract the downloaded package and run npm install in the extracted folder.

4
Configure Claude Desktop

Add the server to your Claude Desktop config:

{
  "mcpServers": {
    "your-server-name": {
      "command": "node",
      "args": ["path/to/your/server/index.js"]
    }
  }
}

Troubleshooting

Common issues and solutions

❌ Tools not showing up in Claude Desktop

  • • Make sure QuickMCP is running (check server status above)
  • • Verify the configuration is correct in Claude Desktop
  • • Restart Claude Desktop completely
  • • Check that port 3001 is not blocked by firewall

🔄 Server status shows offline

  • • Run npm run dev in your QuickMCP directory
  • • Check if port 3001 is already in use
  • • Look at the console for error messages

⚠️ Individual exported servers not working

  • • Make sure you ran npm install in the exported folder
  • • Check that the path in Claude Desktop config is correct
  • • Ensure Node.js is installed and accessible
