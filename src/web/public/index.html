How to Use Your MCP Server

Using with Claude Desktop

🚀 Direct Integration (Recommended)

Use this MCP Server Generator directly as a live MCP server. Your databases and tools will be available in real-time.

1️⃣ Keep This App Running

Make sure this application is running in MCP server mode:

cd /path/to/quickmcp
npm run mcp-server

2️⃣ Configure Claude Desktop

Add this configuration to your Claude Desktop config file:

Location:

  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\\Claude\\claude_desktop_config.json

Add this to your configuration:

{
  "mcpServers": {
    "quickmcp-generator": {
      "command": "node",
      "args": ["/path/to/quickmcp/src/mcp-server.js"]
    }
  }
}

3️⃣ Available Tools in Claude

  • connect_database - Connect to MySQL, PostgreSQL, SQLite, or MSSQL
  • list_tables - List all tables in connected database
  • execute_query - Run custom SQL queries
  • get_table_schema - Get table structure and column info
  • generate_mcp_server - Generate and export MCP servers

4️⃣ Restart Claude Desktop

Restart the Claude Desktop app to load the MCP server. You can then ask Claude to:

  • "Connect to my MySQL database"
  • "Show me all tables in the database"
  • "Run a query to find all users"
  • "Generate an MCP server from this data"

📦 Alternative: Export Static Servers

You can also export individual servers as standalone files:

  1. Click "Export" on any generated server
  2. Extract the ZIP file to a folder
  3. Run npm install in the folder
  4. Configure Claude Desktop to use the exported server.js

💡 Tips

  • Keep this app running in the background for live database access
  • Use absolute paths in the Claude Desktop configuration
  • Check Claude Desktop logs if the server doesn't appear
  • The live server provides more flexibility than exported static servers

Integrating with OpenAI Desktop

📋 Prerequisites

  • OpenAI Desktop app (ChatGPT desktop) installed
  • Node.js installed on your system
  • Your exported MCP server files

1️⃣ Export Your Server

  1. Click the "Export" button for your server
  2. Save the ZIP file to your desired location
  3. Extract the ZIP file to a folder (e.g., ~/mcp-servers/your-server)

2️⃣ Install Dependencies

Navigate to your server folder and install dependencies:

cd ~/mcp-servers/your-server
npm install

3️⃣ Configure OpenAI Desktop

Add your server to OpenAI Desktop's configuration:

Location:

  • macOS: ~/Library/Application Support/OpenAI/mcp_config.json
  • Windows: %APPDATA%\\OpenAI\\mcp_config.json

Add this to your configuration:

{
  "servers": {
    "your-server": {
      "command": "node",
      "args": ["/path/to/your-server/server.js"],
      "enabled": true
    }
  }
}

4️⃣ Restart OpenAI Desktop

Restart the OpenAI Desktop app to load your new MCP server. Your server's capabilities will be available in ChatGPT!

⚠️ Note

OpenAI Desktop MCP support may vary. Check the latest OpenAI documentation for the most current integration steps.

💡 Tips

  • Ensure your server follows the MCP protocol specification
  • Test your server locally first using the Test feature
  • Check application logs for debugging
